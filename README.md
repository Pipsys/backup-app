# Приложение для резервного копирования данных

## Описание
Приложение для резервного копирования файлов и папок с компьютера на внешний жесткий диск с использованием Python и фреймворка Flet.

## Возможности
- Выбор нескольких исходных файлов и папок
- Выбор целевой директории (внешний диск)
- Резервное копирование с сохранением структуры директорий
- Проверка свободного места на диске
- Сравнение файлов по MD5 хешу для определения изменений
- Режим синхронизации в реальном времени
- Логирование операций
- Исключение файлов по паттернам
